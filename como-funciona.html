DÚVIDAS FREQUENTES

Perguntas comuns

Preciso saber programar para usar a Neolab? +
Não. A Neolab foi feita para quem trabalha com planilhas, não para desenvolvedores. Se você sabe usar Google Sheets, já sabe usar a Neolab.
Minha planilha precisa ter um formato específico? +
Não. A IA se adapta à estrutura que você já tem. Colunas com nomes em português, abreviações, formatos diferentes — tudo funciona. Quanto mais organizada, melhores os resultados, mas não é obrigatório.
Meus dados ficam seguros? +
Sim. A Neolab acessa sua planilha apenas para leitura e análise. Não armazenamos seus dados financeiros em nossos servidores. Toda comunicação é criptografada com TLS 1.3.
O bot funciona com qualquer setor? +
Sim, mas é especialmente otimizado para PMEs com controle financeiro, gestão de estoque e análise de vendas. Funciona muito bem para comércio, serviços e e-commerce.
Posso cancelar quando quiser? +
Sim, sem burocracia. Você pode cancelar a qualquer momento pelo próprio painel, sem precisar falar com ninguém.
